# React Calculator

A simple calculator app built with React that performs basic arithmetic operations.

## Features

- Addition, subtraction, multiplication, and division
- Two-input calculator with operator selection
- Division by zero protection
- Clear function to reset inputs

## Getting Started

Install dependencies:
```bash
npm install
```

Run the app:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

## Usage

1. Enter a number in the first input field
2. Enter a number in the second input field
3. Select an operation (+, -, x, /)
4. View the result displayed below
5. Click "clear" to reset
